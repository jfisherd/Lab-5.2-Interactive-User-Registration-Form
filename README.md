

# Lab 5.2, Interactive User Registration Form

## Description
In this lab a responsive registration form is created. Inputs checked for validity, and informs the user if they make an invalid input in one of the fields. Input validity is checked when clicking the Sign Up button and when the user changes focus from an input field to something else.



## Running the program
To view use this lab, open index.html in a web browser<br>
Type in a user name [a-z A-Z _]<br>
Type in an email [a-z A-Z _ .]@[a-z A-Z _ .]<br>
Type in a password []<br>
Type in the password again to confirm it<br>
Click the Sign Up button
